# Loan_Predictions
In this project I have used logistic regression and random forest to determine whether or not a loan will be approved.

I used a fully synthetic dataset from [kaggle]([url](https://www.kaggle.com/datasets/sohailkhan05/loan-risk-prediction)) .

It would be interesting to use a real dataset as I also wanted to explore what is the maximum amount a person can apply for and be predicted approval.
